Quick Start

Learn how to interact with the gen.ggpht API via Virtual Waifu to create images from text prompts.

Biases and content acknowledgment

Despite how impressive being able to turn text into images is, beware to the fact that this model may output content that reinforces or exacerbates societal biases, as well as realistic faces, pornography, violence and possibly copyright protected content. Virtual Waifu runs all external media from gen.ggpht through the NSFW Content Filters API in an attempt to reduce the amount of possibly NSFW content delivered to the users.

3: Using the "gen" feature

About Discord Commands

Interacting with the Virtual Waifu Bot on Discord uses Application.Commands. Application Commands are used to interact with Discord Bot's, and perform other helpful tasks across Discord.

The "gen" command generates a unique image from a text description (known as a Prompt).

Current Issues Across Discord

Currently, Virtual Waifu's "External Content" site (external-content.virtualwaifubot.com) isn't operational, and as such, Discord is caching the rendered results from gen.ggpht's generations. This means that, at time of writing, you are likely to see an irrelevant gen result when multiple Users use Virtual Waifu's "gen" feature on Discord. Stay tuned to find out when this is patched!

Using "gen"

  1. Type (prefix)gen prompt: or select the "gen" command from the slash commands pop-up.
  2. Type a description of the image you want to create in the prompt field.
  3. Press enter, or click the send button, to send the command.
  4. The Virtual Waifu Bot will generate a pop-up asking you to accept the Terms of Service. You must agree to the Terms of Service before any image will be generated.

Respect the Acceptable Usage Policy. The AUP applies wherever the Virtual Waifu Bot is used.


If you don't see a pop-up when typing the /gen command on Discord, try typing the legacy prefix with the command vw!gen, usually this causes a pop-up bug in the Users favor.

Most servers make sure Application Commands only work in specific Channels, or restrict usage by Roles. Application Commands are usually disabled in regular channels like #general.


4: Await the Result

The gen.ggpht API usually takes about 10 seconds to generate a piece.

Generating an image takes approximately 10 seconds (on normal traffic, high traffic times will be higher). Discord Users can make roughly 20 unique Renders an hour, considering caching.

A Render is any action that uses the gen.ggpht API. Renders include using the /gen command to create an image or creating image variations.

Protip: Use gen.ggpht's extensive suffixes, to generate art in a specific style, be it anime, realistic or even extremely vibrant! List of suffixes


5: Suffix List

  1. -lucid is useful for generating vibrantly colored images (some images can cause photosensitive seizures)
  2. -anime is useful for generating anime-themed images.
  3. -smooth is useful in combination with the -anime suffix to get a hand-painted anime style.
  4. -creative generates images that are slightly further from your original prompt.
  5. -beta generates much more "noisy" images.

6: Save Your Image

Click on the image to open it to full size, and then right-click and choose Save image. On mobile, simply tap the image to view it fullscreen, and then tap the download icon in the top right corner. iOS and Android clients may vary.
