How to generate art with Virtual Waifu's "gen" feature!

This quick guide goes over User interactions with the gen.ggpht API.

Disclaimer:

Be advised that any and all prompt text submited via Virtual Waifu's "gen" feature is shared with the gen.ggpht API, and while Virtual Waifu doesn't store or collect any prompt data, gen.ggpht may store, modify, use or reuse any prompt text or Rendered image result submitted or created via their API. Links to the according ToS & Privacy Policy will be linked when published.

Quick Start

Learn how to interact with the gen.ggpht API via Virtual Waifu to create images from text prompts.

A notice from the Developers:
Biases and content acknowledgment

Despite how impressive being able to turn text into images is, beware to the fact that this model may output content that reinforces or exacerbates societal biases, as well as realistic faces, pornography, violence and possibly copyright protected content. Virtual Waifu runs all external media from gen.ggpht through the NSFW Content Filters API in an attempt to reduce the amount of possibly NSFW content delivered to the users.

3: Using the "gen" feature

About Discord Commands

Interacting with the Virtual Waifu Bot on Discord uses Discord's Application.Commands. Application Commands are used to interact with Discord Bot's, and perform other helpful tasks across Discord.

The "gen" command generates a unique image from a text description (known as a Prompt).

Current Issues Across Discord

Currently, Virtual Waifu's "External Content" site (external-content.virtualwaifubot.com) is heavily cached by Discord. This means that, at time of writing, you are likely to see an irrelevant "gen" result when multiple Users use Virtual Waifu's "gen" feature on Discord. Stay tuned to find out when this is patched!


Using "gen"

  1. Type vw!gen prompt: (formatted as vw!gen prompt:anime world), or on Discord, select the "gen" command from the slash commands pop-up.
  2. Type a description of the image you want to create in the prompt field.
  3. Press enter, or click the send button, to send the command.

Respect the Acceptable Usage Policy. The AUP applies wherever the Virtual Waifu Bot is used.

Most Discord servers make sure Application Commands only work in specific Channels, or restrict usage by Roles. Application Commands are usually disabled in regular channels like #general.

If you don't see a pop-up when typing the /gen command on Discord, try typing the legacy prefix with the command (vw!gen). This usually causes a pop-up bug in the Users favor.


4: Await the Result

The gen.ggpht API usually takes about 7-10 seconds to generate a piece (depending on traffic).


Generating an image takes approximately 10 seconds (on normal traffic, high traffic times will be higher). Discord Users can make roughly 20 unique Renders an hour, considering caching.

A Render is any action that uses the gen.ggpht API. Renders include using the /gen command to create an image, or create an image variation.

Protip: Use gen.ggpht's extensive suffixes, to generate art in a specific style, be it anime, realistic or even extremely vibrant! List of suffixes


5: Suffix List

  1. -lucid is useful for generating vibrantly colored images (some images can cause photosensitive seizures).
  2. -anime is useful for generating anime-themed images.
  3. -smooth is useful in combination with other suffixes to get a hand-painted style. Use alone to get a watercolor vibe.
  4. -creative generates images that possibly have nothing to do with your original prompt.
  5. Deprecated: -beta generated much more "noisy" images, usually generating images that had absolutely nothing to do with the original prompt.
  6. -eotw uhh... This kind of replaces the now deprecated -beta. End Of The World vibes tend to come from this.
  7. -photorealistic - it's uhh... more photorealistic..?
  8. -seed (used in combination with an integer from 0-291349430096079 (no commas allowed!)) chose the seed to render the result on. Usually the random one (not adding this suffix) is best. (formatted as gen prompt:anime world -seed 1 to generate "anime world" on seed 1)
  9. There are more seeds coming soon. Stay tuned to find out more!

6: Save Your Image

Click on the image to open it to full size, and then right-click and choose Save image. On mobile, simply tap the image to view it fullscreen, and then tap the download icon in the top right corner. iOS and Android clients may vary.
