Virtual Waifu's commands:

Below is a list of Virtual Waifu's Commands and Features, and a short description of how they are used and work across different platforms!

REMEMBER

Virtual Waifu's legacy prefix (the one that works on Revolt, Crispy-Chat and Discord) is vw!.

We recommend using Slash Commands (/) on Discord for a better experience, but legacy prefixes are still as functional across all platforms!


Public Commands

  • help

    • Shows a helpful list of commands provided by Virtual Waifu!

    Virtual Waifu separates commands by public (any user can use these!), moderator (only moderators/admins can use these!) and private (only the developers can use these!)

  • ping

    • Shows the ping latency. Sometimes you get a nice embed displaying the ping to all the API's, and sometimes you don't. Virtual Waifu runs on two instances.

  • info

    • Displays information about The Application (Developers, API's, website links, etc)

  • ched

    • Summons a random emoji from a pre-defined list.

  • invite

    • Sends you a link to the invite page.

  • avatar

    • Sends your avatar (URL) in the chat.

  • p

    • Parcel below.

  • waifu

    • Summons a random waifu from the waifu.pics API.

  • gen

    • Used to generate art with AI.

    Uses the gen.ggpht API.

    More info

  • play

    • On Discord, this command can play music from Fire.

    On Revolt and Crispy-Chat, this command shows information about the track on Fire.

    Uses the fire-glsp API.

  • server

    • Shows public information about the guild/server you summon the command on.

    For privacy reasons, this only displays information about the guild/server you are currently in.

Moderator Commands

  • lol

    • Become Funny Admin!!1!

  • clear

    • Clears a set amount of messages.

  • warn

    • Warns a User.

  • kick

    • Yeet's a User.

  • ban

    • Permanently Yeet's a User.

Private (Developer) Commands

  • debug

    • Debounces some Debug information that might be helpful to the Developers.

  • togglesafe

    • Gee, I forgot what this does.

  • serverban

    • Bot permanently leaves your server.
