How to setup Virtual Waifu (the way the developers intended)

Below is a simple guide to set up Virtual Waifu the way the developers intended it to be setup.

Invite The Application!

To start off, you're going to want to invite Virtual Waifu.

Head on over to the official invite page, and invite The Application to the Guild/Server of your Platform.

Permissions

Check for the neccessary permissions!

Virtual Waifu requires the following permissions to function on Discord:

  • Kick Members

    • Allows Virtual Waifu to Kick Members

  • Ban Members

    • Allows Virtual Waifu to Ban Members

  • Moderate Members

    • Works in conjunction with both the above (also allows Virtual Waifu to mute, timeout, etc)

  • Read Messages

    • Used for Legacy Prefix compatability

  • Send Messages

    • Used to Send Messages

  • Embed Links

    • Allows Virtual Waifu to embed (as images) the images from waifu.pics and gen.ggpht

  • Attach Files

    • Allows Virtual Waifu to submit (images) from gen.ggpht (that are under 25Mb)

  • Add Reactions

    • Used to React to Messages

  • Use External Emoji

    • Neccessary for the ched feature

  • Connect

    • Connect to Voice Channels

  • Speak

    • Use Microphone in Voice Channels

  • Use Voice Activity

    • Same as above

  • Use External Sounds

    • Play Music from Fire using the fire-glsp API


Virtual Waifu requires the following permissions to function correctly on Revolt:

  • View Channel

    • Allows Virtual Waifu to View Channels in the server (in channels where this permission applies)

  • Send Messages

    • Allows Sending Messages (in channels where this permission applies)

  • Send Embeds

    • Allows Sending Embeds (in channels where this permission applies)

  • Upload Files

    • Allows Uploading Files (in channels where this permission applies)


Virtual Waifu doesn't require any special permissions on Crispy-Chat

The above permissions can either be granted instantly when inviting, or later by roles. If any of the permissions are missing, Virtual Waifu won't be able to work at 100%!

That's it, you're ready to go!
